# Product Overview

This is Danielle Heberling's personal blog website - a modern, fast blog built with Astro and React. The site focuses on technical content related to AWS, serverless architecture, software development, and career advice.

## Key Features

- **Blog Posts**: Technical articles stored as markdown files with frontmatter metadata
- **Speaking Engagements**: Dedicated section for talks and presentations
- **Search Functionality**: Fuzzy search powered by FuseJS
- **Social Integration**: Links to GitHub, LinkedIn, Bluesky, and Discord
- **SEO Optimized**: Automatic sitemap generation, RSS feeds, and OpenGraph images
- **Performance**: Optimized with Jampack for production builds
- **Accessibility**: Built with accessibility best practices using jsx-a11y

## Target Audience

Software engineers, AWS practitioners, and developers interested in serverless architecture and cloud technologies.

## Content Strategy

- **Technical Focus**: AWS, serverless architecture, cloud development
- **Writing Style**: Practical, experience-based content with real-world examples
- **Post Types**: Tutorials, reviews, career advice, conference recaps
- **Frequency**: Regular but quality-focused publishing schedule

## Brand Voice

- **Tone**: Professional yet approachable, sharing lessons learned
- **Perspective**: Practitioner sharing real experiences and insights
- **Community**: Active in AWS and serverless communities
- **Expertise**: Focus on practical cloud development and career growth

## Deployment

The site is deployed on Cloudflare Pages with automatic builds from the main branch.
